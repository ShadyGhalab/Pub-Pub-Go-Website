# Pup Pup Go Website

Static GitHub Pages website for Pup Pup Go.

## Pages

- `index.html`
- `privacy.html`
- `support.html`

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deployment

This repo is intended for GitHub Pages.

```sh
git remote add origin git@github.com:ShadyGhalab/Pub-Pub-Go-Website.git
git branch -M main
git push -u origin main
```
